How to install:
need a special way to enable this mod.

1. download and locate the styles.kvp file, all the modified and larger font text data is inside.
2. open the game install path,  ie D:\Games\The Riftbreaker\packs\startup\
3. use 7Zip or similar zip app to open 01_win_startup.zip file, enter gui path inside. do NOT extract.
4. drag and drop the styles.kvp file in the gui path, confirm to update the zip file.
start the game to see larger text in main gui.

other game menu text are covered in part1
